# Lista-de-Tarefas-em-PYTHON-e-SQL
Sistema de lista de tarefas desenvolvido em Python com integração ao MySQL. Permite cadastrar, listar, editar e excluir tarefas por meio de operações CRUD, utilizando PyODBC para conectar a aplicação ao banco de dados. Projeto criado para praticar Python, SQL e integração com bancos de dados. #Python #SQL #Programacao #Logica
