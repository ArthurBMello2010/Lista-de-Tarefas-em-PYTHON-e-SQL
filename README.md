# Lista de Tarefas em PYTHON e SQL

Sistema de gerenciamento de tarefas desenvolvido em **Python** com integração a um banco de dados **SQL**. O projeto permite cadastrar, visualizar, atualizar e remover tarefas, aplicando operações CRUD (Create, Read, Update e Delete) e demonstrando a comunicação entre Python e SQL.

## 🚀 Funcionalidades

* ➕ Cadastro de novas tarefas
* 📋 Listagem de todas as tarefas
* ✏️ Edição de tarefas existentes
* 🗑️ Exclusão de tarefas
* 💾 Armazenamento permanente em banco de dados SQL

## 🛠️ Tecnologias utilizadas

* Python
* SQL
* PyODBC
* MySQL

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar a integração entre Python e bancos de dados SQL, reforçando conceitos de manipulação de dados, consultas SQL e organização de código. Além disso, faz parte do meu portfólio de estudos em programação e desenvolvimento de software.
